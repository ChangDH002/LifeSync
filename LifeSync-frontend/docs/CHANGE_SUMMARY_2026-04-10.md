# 2026-04-10 변경사항 정리

## 개요

이번 작업에서는 메인 페이지, 마이페이지, 정책 페이지를 중심으로 사용자 화면과 정보 구조를 정비했습니다.

## 주요 변경

### 1. 메인 페이지 개편

- 메인 페이지 레이아웃을 제공된 시안 기준으로 재구성했습니다.
- 히어로 섹션, 주요 기능 카드, 정보 카드 영역을 새로 정리했습니다.
- 기능 카드 높이를 맞춰 시각적인 정렬을 개선했습니다.
- 이모지 기반 표현을 `lucide-react` 아이콘으로 교체했습니다.
- 정보 관련 문구에서 일부 `치매` 표현을 `인지 건강 정보` 중심 표현으로 완화했습니다.

## 2. 마이페이지와 대시보드 통합

- 마이페이지에서 대시보드 정보를 함께 볼 수 있도록 화면을 전면 재구성했습니다.
- 프로필 카드, 설문 안내 배너, 요약 통계 카드, 루틴 진행률, 빠른 이동 카드, 탭형 요약 영역, 최근 활동 영역을 한 페이지에 통합했습니다.
- 기존 단순 안내형 마이페이지를 실제 서비스형 대시보드 구조로 변경했습니다.

## 3. 정책/법적 페이지 추가

- 서비스 이용약관 페이지를 추가했습니다.
- 개인정보 처리방침 페이지를 추가했습니다.
- 의료 고지사항 페이지를 추가했습니다.
- 푸터의 법적 고지 항목을 실제 링크로 연결했습니다.
- 문서는 운영자가 수정 가능한 초안 형태로 작성했고, 실제 사업자 정보 및 개인정보 운영정보는 별도 입력이 필요하도록 표시했습니다.

## 4. 공용 UI 정리

- 공용 `LogoMark` 컴포넌트를 추가해 헤더, 푸터, 로그인 화면의 브랜드 아이콘을 통일했습니다.
- 정책 문서 렌더링용 공용 레이아웃을 추가했습니다.
- 버튼 컴포넌트의 경고 요소를 정리했습니다.
- 문서 메타 정보와 웹폰트 로딩을 `src/app/main.tsx`에서 설정하도록 정리하고, `index.html`은 최소 엔트리 구조로 단순화했습니다.
